MoodCheck

A web app that can be used to analyze the emotions and sentiments of given text input or text content from URLs.

Harnessing one of IBM cloud AI technologies, this web app analyzes the emotional tone and sentiment of text, either from direct input or URL content, providing deep insights into the underlying emotions of the text by indicating a 0-1 scale of the extent of different emotions including sadness, joy, fear, disgust and anger. The backend, built with Node.js, communicates with IBM Watson's API to perform complex natural language processing, while the frontend, designed for simplicity and ease of use, allows users to seamlessly interact with the application to receive real-time emotional analysis of textual content.
